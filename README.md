# ATIKUA – Luxury 3D Restaurant Experience

ATIKUA is an immersive fine-dining web experience designed to combine storytelling, premium visuals, and modern web technologies. The website features cinematic scroll animations, interactive 3D food showcases, a virtual restaurant floor map, and a seamless reservation experience.

## ✨ Features
- 🍽️ Interactive 3D Food Showcase
- 🎬 Cinematic GSAP Scroll Animations
- 🏛️ Luxury Restaurant Floor Experience
- 🪑 Interactive Table Reservation System
- 🌟 Sensory Storytelling Sections
- 📱 Fully Responsive Design
- ⚡ High Performance & Smooth Animations

## 🛠️ Tech Stack
- Next.js 16
- React & TypeScript
- Tailwind CSS
- GSAP & ScrollTrigger
- Framer Motion
- React Three Fiber & Drei
- Three.js

## 🎯 Goal
To create an Awwwards-inspired luxury restaurant experience that blends fine dining with immersive web interactions and cinematic storytelling.
